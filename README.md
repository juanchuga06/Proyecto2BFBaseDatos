## Tema: “Sistema de gestión para franquicia de Barbería Profesional MEN’S”

Este repositorio contiene todos los elementos necesarios para la implementación de una base de datos que soporta el ingreso y salida de datos de un sistema de gestión de una franquicia de barbería profesional. Este sistema fue creado para ser usado en las diferentes sucursales de la franquicia MEN’S. La aplicación fue desarrollada en JavaScript con Node.js y la base de datos se manejó en SQL Server 2025.

## Estructura del proyecto

La estructura de este proyecto es la siguiente. A continuación se detallan los ficheros más importantes:

- **Database:** aquí se encuentran los scripts SQL que incluyen el DDL de la base de datos y una muestra de inserciones en las tablas, por si se desea comenzar a utilizar la base de datos con datos de ejemplo.  
- **Design:** contiene las imágenes en formato `.png` de los modelos entidad–relación, relacional y físico.  
- **Documents:** incluye los archivos `.pdf` del manual y la presentación creada para el proyecto.  
- **barberia_web:** incluye la aplicación web del sistema. 

Ejecutar el programa es bastante sencillo: 
1. Installar Node.js
2. Ubicarse en la carpeta `barbería_web` en la terminal
3. Ejecutar el comando `node server.js`
